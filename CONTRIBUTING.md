# Contributing to MAXX-XMD ⚡

Thank you for your interest in contributing to MAXX-XMD! 🎉  
Every contribution — bug fix, feature, or doc improvement — is welcome.

---

## 🚀 How to Contribute

### 1. Fork the Repo
Click **Fork** at the top right of this page.

### 2. Clone Your Fork
```bash
git clone https://github.com/YOUR_USERNAME/maxxtechxmd.git
cd maxxtechxmd
```

### 3. Install Dependencies
```bash
npm install -g pnpm
pnpm install
```

### 4. Create a Branch
```bash
git checkout -b feat/your-feature-name
```

### 5. Make Your Changes
- Keep commits clean and descriptive
- Use the format: `MAXX-XMD ⚡ — your change description`
- Do NOT expose API keys or session IDs in code

### 6. Push & Open a Pull Request
```bash
git push origin feat/your-feature-name
```
Then open a Pull Request on GitHub targeting the `main` branch.

---

## 🐛 Reporting Bugs

Open an [Issue](https://github.com/Carlymaxx/maxxtechxmd/issues/new/choose) using the **Bug Report** template.  
Include: bot version, Heroku logs, steps to reproduce.

---

## 💡 Suggesting Features

Open an [Issue](https://github.com/Carlymaxx/maxxtechxmd/issues/new/choose) using the **Feature Request** template.

---

## 📋 Code Guidelines

- Node.js 20+
- No breaking changes to the bot engine (`artifacts/` folder)
- Test on a live WhatsApp session before submitting
